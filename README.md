# ⚡ TERMUX BROADCAST ⚡

A Telegram broadcast tool with retry system and live stats.

## Installation

pkg update && pkg upgrade


pkg install git


git clone https://github.com/qwerty987654321my-dot/termux-broadcast-.git


cd termux-broadcast


bash install.sh


python main.py



## Features
- Broadcast messages
- Retry system
- Logs
